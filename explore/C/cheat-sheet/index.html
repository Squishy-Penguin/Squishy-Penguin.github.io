Comment: explain what will happen. Comments will not be compiled (ignored), so they are not required for your code to run, but they make your code easier to understand.

// this is an in-line comment. everything after the '//' will be treated as a comment

/*this is a multiline
comment which means everything within will be treates as
a comment*/

Every statement in C must end with a semicolon ;

printf("Hello, world!); //note the ';' goes at the end of everything
