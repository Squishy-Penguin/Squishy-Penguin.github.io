Downloading a python IDE onto your computer

Having an IDE makes coding easier, especially for a beginner. The simplest (and free) python IDE to use is the Wing 101 Python IDE. There are other options, such as Visual Studio Code (aka VS Code), Spyder, etc.
Of course, you do not have to download an IDE to write a basic python script. You can write a .py (python) script in your preferred way, then run your .py file in command line.
For simplicity sake, I will show you how to download and use Wing 101 to write and run your python code.

You can find the latest version of Wing 101 Python IDE from their official website at: https://wingware.com/downloads/wing-101

To find Wing 101 on your computer, press your windows key and type in Wing 101 into the search bar.

This is where scripts come in handy.
